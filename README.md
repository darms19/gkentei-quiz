# G検定 問題集アプリ

LLM API(Claude / OpenAI / Gemini)で G検定の4択問題を自動生成し、苦手分野を管理できる学習用Webアプリ。

## 起動方法

```sh
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開き、初回に「設定」で使用するAIとAPIキーを設定してください。
スマホから使う場合は `npm run dev -- --host` でLAN公開し、PCのIPアドレスにアクセスします。

## 対応プロバイダ

| プロバイダ | 既定モデル | APIキー取得先 |
|---|---|---|
| Claude (Anthropic) | claude-sonnet-4-6 | https://console.anthropic.com/ |
| OpenAI | gpt-5-mini | https://platform.openai.com/api-keys |
| Gemini (Google) | gemini-2.5-flash | https://aistudio.google.com/apikey |

プロバイダごとにAPIキーとモデル名を保存でき、設定画面でいつでも切り替えられます。

## 機能

- **カテゴリ別出題**: 機械学習・深層学習・数学基礎・AIの歴史・法律・倫理・ビジネス応用
- **難易度選択**: やさしい / 標準 / むずかしい
- **内蔵問題バンク**: 658問(基本は各カテゴリ90問・難易度ごとに30問。探索アルゴリズム・AIガバナンス・強化学習理論などシラバス対応の補強問題を追加)を収録。既定では内蔵問題から先に出題し、使い切ったらAIで生成するため、APIクレジットを節約できる。APIキーなしでも内蔵問題だけで学習可能
- **苦手分野優先モード**: カテゴリ別正解率に基づき、正解率が低い分野ほど高確率で出題
- **ダッシュボード**: カテゴリ別正解率のバーグラフ(低い順)、全体正解率、リセット
- **模擬試験モード**: ミニ(20問/15分)・ハーフ(80問/60分)・フル(160問/120分・本番同等)の3形式。時間制限つきで解答後にまとめて採点し、合格目安(70%)判定・カテゴリ別内訳・全問の解説付き振り返りを表示
- **選択肢シャッフル**: 出題のたびに選択肢の並びをランダム化し、位置で正解を覚えてしまうのを防止
- **復習モード(間隔反復)**: 間違えた問題を自動で復習リストに登録。正解するたびに次回の出題が1日後→3日後→7日後→14日後と延び、すべてクリアで卒業。間違えると最初から
- **フラッシュカード**: 用語集156語をカードで暗記。めくって「覚えた/まだ」で仕分けし、習得状況を保存。カテゴリ絞り込み・未習得のみ出題に対応
- **カンペ(要点シート)**: 苦手カテゴリ(正解率60%未満)の用語・間違えた問題・ブックマークを1枚に集約。カテゴリ/キーワードで絞り込み、PDF保存(印刷)やネイティブ共有・クリップボードへのコピーで持ち出せる
- **用語集モード**: 7カテゴリ・156語のG検定用語をキーワード検索・カテゴリ絞り込みで閲覧でき、各用語からその分野の問題演習にすぐ移れる
- **学習記録**: 連続学習日数(ストリーク)と直近14日の日別解答数をダッシュボードに表示
- **詳細な成績分析**: カテゴリ別に加えて難易度別の正解率、日別正解率の推移グラフを表示
- **ブックマーク**: 気になる問題を解説画面から保存し、一覧からいつでも解き直せる
- **関連用語の表示**: 解説画面に、問題文・解説に登場する用語集の用語を自動で抽出して表示
- **データのエクスポート/インポート**: 成績・解答履歴・ブックマークをJSONで保存し、別の端末・ブラウザに引き継げる(APIキーは含まれない)
- **ダークモード**: ヘッダーのボタンでライト/ダークを切り替え
- **操作性**: どの画面からもホーム/成績/用語集/設定へ移れる下部タブバー、出題中の「中断してホームへ」導線、PCではキーボード(1〜4 または A〜D)で選択肢を回答
- **AI生成問題の事前ストック**: AI出題を使うと、カテゴリ×難易度ごとに5問まで裏で先に生成してストック。2問目以降は待ち時間なしで即時表示され、使った分は自動補充される
- **重複出題の抑制**: 内蔵問題は出題済みを記録、AI生成はカテゴリごとに直近20問の問題文をプロンプトに渡して重複を回避(ストック中の問題とも重複しない)

## 技術構成

- React 18 + Vite + Tailwind CSS v4
- Anthropic Messages API をブラウザから直接呼び出し(`anthropic-dangerous-direct-browser-access` ヘッダー使用)
- データ(APIキー・成績・出題履歴)はすべて localStorage に保存。サーバー不要
- 約8,500行の問題データはカテゴリ別ファイルに分割し、動的import(コード分割)で遅延読み込み。ホーム・設定・用語集などの初期表示を軽量化

## データ検証

問題・用語データの整合性(件数・ID重複・選択肢4つ・正解インデックスの範囲・問題文の重複など)を検証するスクリプトを用意しています。

```sh
npm run validate
```

GitHub Actions(`.github/workflows/validate.yml`)でも push / PR ごとにこの検証とビルドを自動実行します。

## スマホ(出先)から使う — GitHub Pages デプロイ

このリポジトリには GitHub Actions の自動デプロイ設定(`.github/workflows/deploy.yml`)が含まれています。

1. https://github.com/new で **Public** リポジトリを作成(名前は任意。例: `gkentei-quiz`)
2. このフォルダで以下を実行:

   ```sh
   git remote add origin https://github.com/<ユーザー名>/<リポジトリ名>.git
   git push -u origin main
   ```

3. push すると自動でビルド&デプロイされ、数分後に
   `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます
   (もしActionsがPages設定エラーで失敗したら、リポジトリの Settings → Pages → Source を「GitHub Actions」にして再実行)

スマホでURLを開き、ブラウザメニューの「ホーム画面に追加」をするとアプリのように起動できます。
APIキーはビルドや リポジトリには含まれず、各端末で入力して各端末のlocalStorageにのみ保存されます。

## 注意

- 既定モデルは `claude-sonnet-4-6` です。旧既定 `claude-sonnet-4-20250514` は **2026年6月15日に廃止**のため変更しました。以前のバージョンで旧モデルを保存済みの場合は、設定画面からモデルを更新してください。
- APIキーをブラウザに保存する構成のため、**個人の学習用途専用**です。公開サイトとしてデプロイしないでください(キーが第三者に渡る恐れがあります)。
